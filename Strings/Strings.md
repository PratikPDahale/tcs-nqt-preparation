# 📘 TCS NQT Preparation - Strings

This folder contains a collection of important **String-based practice problems** for TCS NQT preparation.
All problems are implemented in **Java**, focusing on logic building and interview readiness.

---

## 🧠 Topics Covered

### 🔤 String Problems

1. Check Whether a String Is a Palindrome
2. Remove vowels from a String
3. Remove spaces from a given string
4. Remove a specific character
5. Remove all characters other than alphabets
6. Reverse a string
7. Sum of numbers in a string
8. Frequency of characters in a string
9. Non-Repeating Character
10. Smallest and Largest Words in a String
11. Check if two strings are anagrams
12. Sort string of characters
13. Convert characters of a string to opposite case
14. Count words in a given string
15. Encrypt the string
16. Remove duplicates from a string
17. Check if a string is a substring of another
18. Reverse words in a string
19. Count Distinct Subsequences
20. Common subsequence in two strings
21. String with wildcard characters
22. Longest Repeating Subsequence
23. Check if one string is a subsequence of another
